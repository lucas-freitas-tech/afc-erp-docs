# 📘 AFC ERP — Documentação Oficial

Bem-vindo ao repositório de documentação do **AFC ERP**, sistema corporativo modular que evolui por releases incrementais, com foco em segurança, facilidade de gestão e expansão contínua.

Aqui você encontra toda a documentação funcional e técnica organizada por versão do produto.
